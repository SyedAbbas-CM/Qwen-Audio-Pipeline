# Final Devlog Script

## HOOK

This is my game. One person, no budget. Utilizing tools like Claude Code and my own custom codex to build this out—let’s see how far I can push it.

## Marker 4 - Intro

This is a 2D bullet hell, open-world type of game. It’s got dungeons, crafting, traps, and bosses. I’ve always wanted to make a game like this, so I’m finally just doing it.

## Marker 5 - Physics

The physics system took a while to fine-tune. We’ve got four classes of weight now—light to super heavy. Minions and bosses have their own weight ranges, and the player does too. It’s not purely a bullet hell; there’s actual weight to how everything moves.

## Marker 6 - VFX

I also added some cool trailing dash lines. I think they look pretty neat.

## Marker 10 - The Dungeon

This is the core of the game for now: a dungeon. It’s got a bunch of traps, corridors, and rooms. I'm going for a full spooky vibe—bloodstains, skulls, skeletons. The usual.

## Marker 14 - Lighting

The coolest thing I’ve added is dynamic lighting. This dungeon is lit darkly on purpose. I’m thinking about doing "inverse light" dungeons later, where enemies blend into the light instead of the shadows—kind of like that one shinobi scene from Samurai Jack, you know?

## Marker 11 - Treasure

This is a treasure room. You can get extra health pots or some rarer drops here.

## Marker 12 - Testing

Right now I’m using a character that’s done this dungeon 4 or 5 times, so they’re pretty darn strong.

## Marker 13 - Procedural Gen

Procedural generation here involves corridor and room logic. Instead of a bunch of random rooms, we have specific types of corridors to make it feel more maze-like.

## Marker 15 - Length

This goes on for a really long time. I might trim the length... maybe.

## Marker 16 - Elites

Here are some elite enemies. They’re like mini-bosses—they drop better gear and have a more varied and dangerous attack pattern.

## Marker 17 - Balancing

This guy is spamming way too many bombs... I definitely need to fix that.

## Marker 18 - Stats and Armor

Also, a lot of these traps should be getting through my armor and dealing pure damage. There’s a stat called Vitality that gives you defense and HP regen, but it’s kind of overpowered right now.

## Marker 19 - Rare Rooms

This is a rarer decorated type of room. It can have an enemy type above an elite—they might as well be bosses. They're rare, though; probably won't see one here.

## Marker 20 - Progression

We’ve got descending floors for dungeons and it keeps going until you get to the final boss. Maybe I’ll make an infinite dungeon too that just loops; that could be cool.

## Marker 21 - Hallways

These corridors actually have enemies in them that patrol the halls. I’m looking to add a sneaking system too, which could be pretty interesting.

## Marker 22 - Inventory and Stats

Here's the inventory system. It’s a bit buggy right now. There are spells, items, and an enchanting system that technically isn't supposed to be here yet, but it’s in for testing purposes. To the right is the stat system where you can apply points to things like Agility and Power.

## Marker 23 - Items and Movesets

Different item types have different movesets on Shift and Spacebar. The shortbow lets you run if you hold Shift and releases a triple shot that pushes you back so you can retreat, while the longsword lets you dash into enemies and push them back.

## Marker 26 - The Boss

So, here’s the dungeon boss. He has explicit phases and special RNG drops. The plan was a bullet pattern system based on fractals—mathematically beautiful chaos. Right now he’s just doing random stuff and I genuinely don't know why... so, yeah. Work in progress.

## Marker 28 - Height System

Top-down games can feel really flat. Everything is on the same level. I’ve been working with shaders and art to give the world actual elevation. I’m going to use this for fortified areas with archers where you’d need to climb uphill to deal with them.

## Marker 29 - Outro

That’s about it, honestly. There are plenty of bugs and the gameplay loop isn't fully tight yet, but the bones are good. If you like this sort of thing, subscribe—it actually helps out more than you'd think. Catch you in the next one.
